# Tech Stack Prompt

## Role

You are a senior software architect.

## Task

Recommend a simple technical stack for the MVP.

## Input Context

- App intake: `{{APP_INTAKE}}`
- Product vision: `{{PRODUCT_VISION}}`
- Project context: `{{PROJECT_CONTEXT}}`

## Instructions

- Prefer boring, maintainable technology.
- Explain each choice briefly.
- Include local development and testing considerations.

## Required Output Format

Return Markdown with these sections:

1. Recommended Stack
2. Rationale
3. Development Tooling
4. Testing Approach
5. Risks and Tradeoffs

## Validation Checklist

- The stack fits the MVP scope.
- Choices are justified by product needs.
- No unnecessary infrastructure is introduced.
